# テンプレ自己言及の最終検証

> 対象フロー: 新規・既存とも ｜ 種別: auto

ブロック本体は「テンプレ導入ブロックの削除」タスクで削除済み。ここでは **/init・recommender・improver のいずれかの編集で本文へ入った（または improver が復活させた）テンプレ自己言及が、CLAUDE.md 全体に残存していないか**を最終確認する（全文 grep スイープ）。

- grep 相当で次が CLAUDE.md に残っていないか検査する：`TEMPLATE-ONBOARDING` マーカトークン／セットアップ前提の自己言及文（「これは汎用テンプレートである」「まず最初に `/project-setup` を実行」等の、テンプレートそのものへの自己言及・セットアップ前提の案内文）。
- **見つかった場合は当該箇所を除去（または当該プロジェクト固有の説明へ書き換え）し、再度 grep して残っていないことを確認する**。
  - **除去対象**：テンプレートそのものへの自己言及・セットアップ前提の導入文。
  - **保持対象**：「プロジェクト設定」節、標準開発ワークフロー・各規則の説明、Hooks・プラグイン説明、コーディング規約など、適合後も有用な恒久情報（**過剰除去しない**）。

> improver（「CLAUDE.md 改善キュレーション」タスク）が CLAUDE.md を再編集して自己言及を復活させうるため、この最終検証は **必ず improver の後**に置く（ブロック本体の削除は罠回避のため「テンプレ導入ブロックの削除」タスク＝`/init` 前で済ませている）。

`【完了条件】grep 相当で CLAUDE.md に TEMPLATE-ONBOARDING マーカおよびセットアップ前提の自己言及文が 1 つも残っていないことを提示（残存があれば除去後の再 grep 結果も提示）。`
