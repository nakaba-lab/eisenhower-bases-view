# 用語集

業務用語（日本語）と、コード上で使う英語識別子の対訳。**同一概念に複数の英語名（訳語のブレ）を生まない**ための正。新しい業務概念を命名したらこの表に追記してから使う（ローマ字識別子は禁止＝`post_edit.py` が検知。英語名の正はここに置く）。

| 用語（日本語） | 英語名（識別子） | 使用箇所・備考 |
|----------------|------------------|----------------|
| 象限 | quadrant | 緊急×重要の 4 区分（Do/Schedule/Delegate/Delete）。`classifyQuadrant` |
| 緊急度（軸） | urgency (axis) | 横軸。boolean プロパティ（v1） |
| 重要度（軸） | importance (axis) | 縦軸。boolean プロパティ（v1） |
| 軸プロパティ | axis property | 象限算出に使う `note.*` の frontmatter プロパティ |
| 未分類ゾーン | unclassified zone | 軸欠損（absent）ノートの置き場。ドロップ不可 |
| 欠損 / 未定義 | absent | プロパティ未定義。`false` とは区別する |
| 書き戻し | write-back | ドラッグで frontmatter を更新する操作。`processFrontMatter` |
| 楽観的更新 | optimistic update | 書き込み確定前に UI を先行更新し、失敗時ロールバック |
| エントリ | entry | Bases が返す 1 ノート分のデータ（`BasesEntry`） |
| ビュー設定 | view options / settings | Bases ビュー options（主）＋プラグイン設定（デフォルト） |
| 象限ラベル | quadrant label | 各象限の見出し。カスタム空は言語既定にフォールバック（F6）。`quadrantLabels` |
| 象限色（アクセント色） | quadrant color / accent color | 象限の左ボーダー色。空はテーマ既定（`--interactive-accent`）追従（F6）。`quadrantColors` |
| 表示言語 | language | UI 文言・既定象限ラベルの言語。`auto`（Obsidian 追従）/`en`/`ja`（F6）。`resolveLanguage` |
| 表示情報 | presentation | UI へ渡す解決済みラベル/色/言語文言の束（F6）。`MatrixPresentation` |
| 滞留 | stagnation / stagnant | 最終更新（`mtime`）から一定日数を超えて動いていない状態。バッジで可視化（F7）。`evaluateStagnation`・`stagnant`・`stagnantDays` |
| 滞留しきい値 | stagnation threshold | 滞留とみなす日数（既定 14・0 でオフ）。ビュー options 主＋設定既定のハイブリッド（F7）。`stagnationThresholdDays` |
